Services

Data Analysis & Visualizations

Can analyze a given dataset using core Python with data analytics tools such as NumPy, Pandas, and Matplotlib. Also able to construct interactive data visualizations using core web development technologies (HTML, CSS, and Javascript) which include Javascript libraries such as Leaflet and D3

Database Development & Querying

Able to design and create database structures and queries using MySQL, MongoDB, and SQLite in order to organize data into easily retrievable formats. Can also manipulate data as well as perform complex queries and simple analyses using Microsoft SQL Server.

Full Stack Web Development

Able to incorporate technologies and techniques associated with both front-end and back-end web development in order to produce a website including database development and design, object-oriented programming, application architecture, and server hosting.
