HR Data Analytics Portal

This web app is a data analytics portal prototype for the HR Department of Dental Magic, Inc. -- a fictitious company based in Boston, MA, that offers both dental services and dental-related products. My Weather Forecaster

Weather forecasts by location courtesy of NOAA...

Go To Website Go To Code

This web app provides 5-day weather forecasts based on the location of the user. Users that register have the ability to personalize weather forecasts by providing up to 5 US locations (zipcodes) of their choice. Three APIs are consumed for weather and location information, and an "admin" utility is incorporated within the site for administration purposes.

One unique aspect to point out about this NOAA API is that it outputs XML data (NOAA actually has more than one API available). Within this app, XML data is parsed using the Python implementation of the ElementTree library.

Front-End Development: HTML5, CSS3, Bootstrap 4, Javascript
Back-End Development: Python v3.7
Data Storage: Azure DB-PostgreSQL, Azure Blob Storage
Website Framework: Django v2.2
Host Server: Azure App Service
Other Technologies Used: NOAA's National Weather Service Forecast API, Ipstack.com API, Google Maps API, ElementTree, Visual Studio Code
